# 💰 Personal Expense Tracker CLI

A powerful and user-friendly command-line application built with Python to manage personal finances, track transactions, and control monthly budgets efficiently.

---

## 🎥 Demo

📺 Watch the project in action:
👉 https://youtu.be/gCAWCZ4n5T4

---

## 🚀 Features

* ➕ Add income and expense transactions
* 📋 View all transactions with exact dates
* ❌ Delete transactions by ID
* 🎯 Set monthly budgets for categories
* 📊 View budget summaries (spent & remaining)
* 📁 Export all transactions to CSV (using pandas)
* 🧠 Input validation for better user experience
* 🖥️ Simple and intuitive CLI interface

---

## 🛠️ Technologies Used

* 🐍 Python
* 📊 Pandas
* 📂 File Handling (CSV)
* 🖥️ Command Line Interface (CLI)
